# Repositório: A4_PS

Projeto de atividade da disciplina de Processo de Software - 3º semestre de Engenharia de Software pela UNOESTE.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Bootstrap Icons

## Descrição

Site desenvolvido como parte da Atividade 4 do curso. O projeto usa:

- HTML5
- CSS personalizado em `css/style.css`
- JavaScript em `js/script.js`
- Bootstrap 5 via CDN
- Ícones Bootstrap Icons via CDN

## Como executar

- Abra o arquivo `index.html` no navegador web.

## Estrutura do projeto

```bash
A4_PS/
├──assets/
├──css/
│   └──style.css
├──js/
│   └──script.js
├──pages/
├──.gitignore
├──LICENSE
├──README.md
└──index.html
```

## Dependências

O site carrega Bootstrap e Bootstrap Icons diretamente pelos links CDN presentes na head do HTML5.
